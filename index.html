주요 기능

Features

  • 클립보드 자동 캡처 - 게임에서 아이템 복사시 자동 인식
  • Auto clipboard capture - automatically detects copied items in-game
  • 로컬 저장 - 외부 서버 없이 개인정보 보호
  • Local storage - protects personal info without external server
  • 실시간 오버레이 - 게임 화면에 아이템 표시
  • Real-time overlay - displays items on game screen
  • 아이템 통계 - 일별/주별 캡처 현황
  • Item statistics - daily/weekly capture stats
